# JavaScript-Drum-Kit
Play the drums with your keyboard and mouse click

[![Greenkeeper badge](https://badges.greenkeeper.io/sethbergman/JavaScript-Drum-Kit.svg)](https://greenkeeper.io/)

### Install dependencies

```bash
npm i
# or use yarn
yarn
```

### Start the application

```bash
npm start
# or
yarn start
```

### Jam out :smiley:

Have fun!
